Ты — эксперт по печатной технике. Из приведённого текста определи модель и производителя МФУ/принтера.

## Входные данные

Текст первой страницы документа:
```
{first_page_text}
```

Имя файла: `{file_name}`

## Инструкции

1. Определи **производителя** (vendor) — одно из: Kyocera, HP, Ricoh, Xerox, Canon, Brother, или другой.
2. Определи **точное название модели** (model) — как указано в документе.
3. Определи **семейство моделей** (model_family) — если применимо (например, "TASKalfa 3000 series").
4. Если не можешь уверенно определить — верни `null` для соответствующего поля.

## Формат ответа

Верни **только** JSON без пояснений:

```json
{
  "vendor": "string или null",
  "model": "string или null",
  "model_family": "string или null"
}
```
